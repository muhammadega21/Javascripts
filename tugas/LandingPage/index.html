Kelas Kafekoding

#HTML & CSS

HTML Merupakan bahasa markup untuk membuat halaman website. Untuk mengatur halaman website tersebut dibutuhkan CSS agar tampilan web terlihat rapi dan cantik.

#Javascript

JavaScript adalah suatu bahasa pemrograman tingkat tinggi dan dinamis. Javascript mampu menampilkan web menjadi lebih interaktif dan menarik dengan konten dinamisnya.

#PHP

PHP adalah bahasa pemrograman yang ada di sisi server. Bahasa ini digunakan untuk mengelola konten dinamis, database, session tracking, dan bahkan keseluruhan situs e-commerce.

#Database

Database adalah kumpulan informasi yang disimpan secara sistematik dalam komputer sehingga kemudian dapat diperiksa dengan mudah melalui program komputer tertentu.

#Design Grafis

Design grafis adalah proses komunikasi menggunakan elemen visual, seperti tulisan, bentuk, dan gambar yang dimaksudkan untuk menciptakan persepsi akan suatu pesan yang disampaikan.
